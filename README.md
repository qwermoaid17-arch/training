# 🛡️ Core User Management System

A robust, terminal-based User Authentication and Management System built entirely with Python. This project serves as a comprehensive demonstration of Back-end logic, Data Validation, and Database Management.

## ✨ Key Features
* **User Registration & Login:** Secure authentication process.
* **Data Validation:** Utilizes Regular Expressions (RegEx) to ensure emails and passwords meet strict security standards.
* **Data Persistence:** Integrated with SQLite to store user data permanently.
* **Error Handling:** Built with Try/Except blocks to prevent system crashes.

## 🛠️ Technologies Used
* Python 3
* SQLite3 (Database)
* OOP (Object-Oriented Programming)
